### 📄 Descrizione
iAnime è una web app moderna che unisce la passione per gli anime con le potenzialità del web. Offre un'esperienza completa per gli appassionati, dalla scoperta di nuovi titoli alla condivisione con la community, fino all’acquisto di merchandising.

---

### 📚 Descrizione Completa
La applicazione si chiama "iAnime" ti permette di visualizzare i dettagli di un anime tra quali l'immagine di copertina, il nome dell'anime, la descrizione dell'anime, la categoria, quanto e stato valutato, le piattaforme dove vedere l'anime (in base al paese in cui ti trovi, hanno un link ipertestuale alla pagina della piattaforma riferita all'anime) e la lista di community avente l'hashtag dell'anime, l'utente per fare ciò inserisce in una barra di ricerca il nome dell'anime interessato e  gli verrà mostrata una lista di risultati di ricerca nella quale l'utente può scegliere l'anime desiderato (dashboard home).

Il programma ti permette di creare un account per la applicazione inserendo una mail, una password e un nome utente obbligatori, inoltre nella pagina di login puoi decidere di cambiare password e la piattaforma ti invierà una mail che ti rindirizzerà ad una pagina per cambiare password. una volta fatto il login l'utente viene rindirizzato alla dashboard home. l'utente può modificare i dettagli del suo profilo tramite una dashboard apposita. da questa dashboard l'utente può modificare il nome utente (ogni 5 giorni, univoco per ogni utente), la sua biografia, la sua immagine di profilo, il colore di display del nome utente (premium, l'utente può fare una preview), la cornice della immagine di profilo (premium, l'utente può fare una preview). da questa pagina l'utente si può spostare alle impostazioni dell'account, che si dividono in generali, privacy e sicurezza e premium:

In generale l'utente può modificare queste impostazioni:
- la mail (solo dopo aver attivato il 2fa), 
- la password (solo dopo aver attivato il 2fa)
- eliminare account(pagina apposita di conferma, con tanto di invio mail)

In privacy e sicurezza l'utente può modificare queste impostazioni:
- attivare il 2fa (associando un numero di telefono, una mail di recupero o tramite una app autenticatore)
- decidere chi può invitarlo nelle community (tutti, solo amici, nessuno).
- decidere se accettare i messaggi degli sconosciuti (si/no).

In premium l'utente può modificare le seguenti impostazioni:
- Acquistare l'abbonamento premium (5.99$/mese, con pagina di conferma e se non già settato pagina per settare il metodo di pagamento, le funzionalità premium sono segnate con la parola chiave "premium")
- modificare/aggiungere metodi di pagamento (carta credito/debito o prepagata)

L'utente può anche accedere alle impostazioni della applicazione, che si suddividono per personalizzazione, notifiche, accessibilità, audio.

In personalizzazione l'utente può modificare queste impostazioni:
- Il tema dell'applicazione (Vari temi monocolore + temi designer per gli utenti premium, l'utente può fare una preview dei design premium)
- Scegliere la modalità di visualizzazione (chiaro/scuro)

In notifiche l'utente può modificare queste impostazioni:
- Se menzionato in una community (ON/OFF, in caso di ON mostra una lista di community modificabile)
- Per richieste di amicizia (ON/OFF)
- Per notifiche di sistema (ON/OFF)
- Persone limitate (lista di utenti modificabile)

In accessibilità l'utente può modificare queste impostazioni:
- Modalità daltonico (con tutti i tipi di daltonismo, anche i non daltonici)
- Modalità alto contrasto (ON/OFF)
- Dimensione testo (da 0.25 a 1 VW)

In audio l'utente può modificare queste impostazioni:
- Dispositivi di entrata (lista di dispositivi audio d'entrata)
- Dispositivi di uscita (lista di dispositivi audio d'uscita )
- Volume (da 0 a 100)

Inoltre l'utente può socializzare con gli altri utenti tramite chat private o le community, per socializzare l'utente può accedere ad una dashboard, nella quale può cercare utenti tramite il loro nome utente, accettare richieste di amicizia, chattare con gli utenti e chattare nelle community. per inizializzare una chat con un altro utente il primo utente deve cercare l'utente desiderato tramite il nome utente, la applicazione gli mostrerà una lista di utenti e l'utente può cliccare sull'utente desiderato, una volta fatto ciò l'utente vedrà i dettagli del profilo dell'utente desiderato insieme ad un tasto per inviargli una richiesta di amicizia, l'utente desiderato può accettare la richiesta dalla dashboard apposita oppure rifiutarla. Inoltre dai dettagli profilo di un'utente puoi decidere di bloccare un altro utente, un utente bloccato da un altro utente non riesce più a cercare l'altro utente e i messaggi che gli scrive non gli arrivano. nella dashboard delle community l'utente può scegliere

Una community è un gruppo nella quale vari utenti possono chattare tra di loro, una community è composta da un admin, una lista di utenti normali, una lista di ruoli composta da una lista di utenti normali, una lista di categorie (alcune strettamente collegate ad un anime, ex. naruto, jujutsukaisen), una lista di gruppi di canali composti da un nome e una lista di canali (10 persone max), i canali possono essere chat testuali o chat vocali. 
un admin può fare varie cose in una community:
- Cacciare utenti dalla community
- Eliminare per tutti un messaggio di un'altro utente
- Mutare un utente (per canale/lista canali e tempo)
- Modificare, aggiungere e rimuovere i ruoli e i loro permessi (quelli elencati qui + in quali canali possono interagire e il colore del loro nome in display nella community)
- Modificare, aggiungere e rimuovere e aggiungere i canali
- Modificare, aggiungere e rimuovere gli hastag del canale
- Modificare, aggiungere e rimuovere i gruppi di canali
- Pinnare messaggi

un utente normale può fare queste cose in una community:
- Scrivere in un canale testuale in base ai suoi permessi
- Accedere in un canale testuale in base ai suoi permessi
- Lasciare la community
- Segnalare messaggi e la community

### 🛠️ Tecnologie Utilizzate

#### Frontend:
- **React.js** – Libreria per la creazione dell'interfaccia utente
- **React Router** – Gestione delle rotte
- **Axios** – Chiamate HTTP verso API esterne
- **Socket.io-client** – Chat real-time

#### Backend:
- **Node.js** – Runtime JavaScript lato server
- **Express.js** – Framework backend veloce e leggero
- **Socket.io** – Comunicazione in tempo reale
- **MySQL** – Database relazionale per la gestione dei dati applicativi
- **mysql2** – Driver MySQL per query SQL raw da Node.js

#### API esterne:
- **Jikan / Anilist / Kitsu** – Ricerca anime
- **JustWatch API** – Verifica disponibilità streaming
- **(Opzionale)**: API di e-commerce per merchandising
